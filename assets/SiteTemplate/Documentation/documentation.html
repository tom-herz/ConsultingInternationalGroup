Javascript

It's possible to change the JS settings by extending the defaults from the template, that way is so much easier to update the theme changing only one JS file.
Here are a few samples on how to do that:


Changing Navbar Colleapse

TO (HTML): 
<div class="collapse navbar-collapse">
(Javascript): 
$('.collapse').not(".navbar-collapse").collapse();


Manually Initializing a Plugin

Some of the plugins/elements are initialized automatically, if you want to disabled that and call the plugin via Javascript, follow these steps:

FROM (HTML): 
<div class="owl-carousel" data-plugin-options='{"items": 6, "singleItem": false, "autoPlay": true}'>
TO (HTML): 
<div class="owl-carousel manual" id="owl-example">
(Javascript): 
$("#gallery").owlCarousel({
 items : 4,
 itemsDesktop : [1199,3],
 itemsDesktopSmall : [979,3]
});

Effects

You can add some Animations / Transitions to the elements which fires when an element scrolls into view, it's pretty simple to use and you can add it in any element.
 

data-story-tale-item This is used for marking a text zone, this is used for Stricly TEXT Only, to achieve an awesome Text Story Display

data-story-tale-container This is used for marking a block zone. This can be used anywhere.

Usage:

<div data-story-tale="rotateIn,rotateInDownLeft,rotateInDownRight,rotateInUpLeft,rotateInUpRight">
...
</div>

Heads up: The Attribute data-appear-animation-delay is Optional and is set in Miliseconds.

The animation can be set to:


  • rotateIn
  • rotateInDownLeft
  • rotateInDownRight
  • rotateInUpLeft
  • rotateInUpRight
  • fadeInUp
  • fadeInRight
  • fadeInDown
  • fadeInLeft
  • fadeInUpBig
  • fadeInRightBig
  • fadeInDownBig
  • fadeInLeftBig
  • bounceInUp
  • bounceInRight
  • bounceInDown
  • bounceInLeft
  • flipInX
  • flipInY
  • rotateIn
  • rotateInDownLeft
  • rotateInDownRight
  • rotateInUpLeft
  • rotateInUpRight
